# Preface
command line tools are defined in `.bash_tools/`. `$DOTFILES` and `$XDG_[CACHE|CONFIG|DATA]_HOME` are honored, but default to `$HOME`.

# Installation

1. Clone this repository
2. Change into its directory
3. Execute `initialize.sh`
